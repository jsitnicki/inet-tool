"Usage: inet-tool {unload,load,info,list,bind,unbind,register,unregister,scm_serve,scm_register} ...\n"
"Manages INET_LOOKUP eBPF program, responsible for programmable socket dispatch.\n"
"\n"
"  inet-tool load\n"
"      Loads INET_LOOKUP program.\n"
"\n"
"  inet-tool unload\n"
"      Unloads INET_LOOKUP program.\n"
"\n"
"  inet-tool info\n"
"      Prints info about INET_LOOKUP program.\n"
"\n"
"  inet-tool list\n"
"      Prints services and bindings.\n"
"\n"
"  inet-tool bind [protocol] [ip:port] [service]\n"
"      Create a binding between protocol,ip:port selector and a service.\n"
"      For example \"inet-tool bind 6 0.0.0.0:80 http-service\"\n"
"\n"
"  inet-tool unbind [protocol] [ip:port]\n"
"      Removes a binding.\n"
"\n"
"  inet-tool register [service]\n"
"      Allocates a slot for a service.\n"
"\n"
"  inet-tool register [service...] -- [command]\n"
"      Registers sockets passed down from parent with socket activation,\n"
"      and launches a command. This is useful wrapper for socket activation.\n"
"\n"
"  inet-tool unregister [service]\n"
"      Deregisters a service.\n"
"\n"
"  inet-tool scm_serve --unix=UNIX_SOCKET\n"
"      Creates unix domain socket and waits for SCM_RIGHTS sockets on it.\n"
"      This is useful service for socket activation. Default path is @inet-scm-service.\n"
"\n"
"  inet-tool scm_register --unix=UNIX_SOCKET [service...] -- [command]\n"
"      Registers sockets passed down from parent with socket activation,\n"
"      and launches a command. This is useful wrapper for socket activation.\n"
